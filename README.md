# Customer Segmentation Using K-Means Clustering

## Objective
Group customers into different segments based on spending behavior and income.

## Features
- Customer Behavior Analysis
- K-Means Clustering
- Data Visualization
- Customer Group Identification

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
Mall Customers Dataset

## Output
Cluster visualization showing different customer segments.
